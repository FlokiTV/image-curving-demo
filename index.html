Curved image

How to curve an image using JavaScript and a <canvas> element.

This is a demo / proof of concept built for my blog post Drawing a curved image with a HTML5 canvas and JavaScript on Itsy Bitsy Bytes. You can read the post for more info or check out the source on GitHub: Image Curving Demo. Simply select different options and press on the Warp Image button to see the result!
